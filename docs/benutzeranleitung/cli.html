Command Line Interface (CLI)¶

Overview¶

This section provides instructions for using the command line interface (CLI) of the Payload Computer Drone Project.

Getting Started¶

To access the CLI, open your terminal and navigate to the project directory.

Basic Commands¶

python start_scripts\start_raspi_px4.py -h

This will give you further instructions but basicly only the mission_file.json is needed.
